# TCL TV OS 灵悉 UI 还原版 - 交互 Demo 运行指南

本 Demo 根据中国区真实的 5 张电视系统设置与控制面板截图进行像素级重绘与高保真还原。

## 1. 运行与操作说明

### 打开方式
直接双击 [index.html](file:///Users/eric/Downloads/需求原型设计/ProductPrototypeKit/prototype/index.html) 即可在现代浏览器（Chrome、Safari、Edge）中离线打开。

### 遥控器键盘模拟映射
| 真实遥控器 | 键盘按键 | 操作说明 |
| :--- | :--- | :--- |
| **方向上/下/左/右** | `ArrowUp/Down/Left/Right` | 移动屏幕焦点；当滑块（Slider）或选项（Picker）聚焦时，**左/右**键可调整对应数值/切换选项。 |
| **确认键 (Enter)** | `Enter` | 激活聚焦按钮、切换开关、或选择列表/弹窗选项。 |
| **返回键 (Back)** | `Escape` | 关闭弹窗，或逐级返回上级菜单。 |
| **快捷工具/控制面板** | `KeyC` | 在桌面模式下，按 `C` 键呼出/关闭左侧快捷控制中心（Image 1 浮层）。 |

## 2. 真实还原的交互看点

### 看点 A：桌面 Launcher 与快捷控制中心切换（图 1 还原）
- 界面初次打开显示 TCL 智慧桌面。
- 键盘按下 `C` 键，左侧平滑滑出 **“快捷控制中心”**，包含天气时钟、亮度滑块、网络、音频输出、蓝牙状态以及环境感应开关。

### 看点 B：侧边类别栏“收缩-展开”微交互（图 2 与 图 3 对比）
- 在控制中心中聚焦并点击 **“全部设置”** ➔ 呼出主设置菜单。
- 此时焦点在最左侧“图像、声音”类别栏时，侧边栏宽度为 `250px` 并显示中文（图 2）。
- 向右按 **右方向键** 将焦点移至中间设置项（如图效模式）时，侧边栏宽度自适应缩窄为 `90px`，中文字体隐藏，仅留下高保真图标（图 3），释放了内容显示面积。

### 看点 C：图效协议联动（图 2 与 图 3 对比）
- 聚焦在“图效模式”，按左右方向键在“FILMMAKER”、“标准”、“杜比视界IQ”之间切换。
- 当图效为其他时，顶部协议字样显示 **SDR**（图 2）；当切换至 **“杜比视界IQ”** 时，顶部标签无缝变更为带高亮标示的 **Dolby Vision IQ**（图 3）。

### 看点 D：高级设置级联面板展开（图 4 与 图 5 还原）
- 下移焦点并点击“高级设置”下的 **“亮度”** ➔ 中间栏切换为亮度的参数群，包含自动 HDR、黑电平、伽马选项（图 4）。
- 按 Esc 键退回，下移聚焦并点击 **“色彩”** ➔ 中间栏切换为色彩参数群，包含饱和度、色调、色温调节滑块（图 5）。
- 在“亮度”菜单中，可以演示新增的“智能低蓝光护眼模式”和“防蓝光强度”的级联解锁交互。
