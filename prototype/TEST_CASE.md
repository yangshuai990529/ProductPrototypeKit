# 测试用例库 (TEST CASE) — 灵悉 UI 还原版

## 1. 快捷控制中心测试

### TC-01: 控制中心唤出与滑入动画
- **前置条件**：界面处于系统桌面。
- **测试步骤**：
  1. 按下键盘上的 **C** 键（模拟遥控器快捷键）。
- **预期结果**：
  - 桌面被暗色半透明滤镜覆盖并轻微模糊（高斯模糊效果）。
  - 左侧控制中心平滑向右滑出，呈现 16:53、天气标签以及快速项列表。
  - 默认焦点高亮停留在“全部设置”卡片上。

### TC-02: 亮度快捷滑块增减
- **前置条件**：控制中心已展开。
- **测试步骤**：
  1. 下移焦点聚焦到“🔆 亮度”卡片。
  2. 按遥控器 **左方向键**，再按 **右方向键**。
- **预期结果**：
  - 按左方向键时，亮度数值（30）递减；按右键时，数值递增。
  - 填充长条以 5% 的步长进行伸缩。

## 2. 图像设置与级联收缩测试

### TC-03: 侧边分类栏自动收缩折叠
- **前置条件**：已点击全部设置，进入系统设置页。
- **测试步骤**：
  1. 初始状态下聚焦在左侧分类栏“图像”项，观察侧边栏宽度。
  2. 按下遥控器 **右方向键**，将焦点移到中间一列的“图效模式”项，再次观察侧边栏宽度。
- **预期结果**：
  - 初始时侧边栏宽度为展开状态（`250px`），完整展示汉字“图像”。
  - 焦点右移后，侧边栏宽度自动平滑缩减至折叠状态（`90px`），汉字文字淡出消失，仅保留“🖼”图标。

### TC-04: 信源协议 Dolby/SDR 切换联动
- **前置条件**：焦点位于中间一列“图效模式”上，当前值为 “FILMMAKER”。
- **测试步骤**：
  1. 按遥控器 **右键** 将图效循环切换至 “杜比视界IQ”。
  2. 观察中间栏顶部的协议标志变化。
  3. 再次按右键切换到其他图效。
- **预期结果**：
  - 切换为“杜比视界IQ”后，顶部 SDR 字符自动变更为高亮蓝色的 **Dolby Vision IQ**，并弹出 Toast 提示。
  - 切换回其他图效后，标识自动变回 **SDR**。

## 3. 高级二级面板调节测试

### TC-05: 亮度高级设置面板切换
- **前置条件**：处于图像一级设置页。
- **测试步骤**：
  1. 下移焦点并点击“高级设置”下的 **“亮度”**。
- **预期结果**：
  - 中间一栏的设置项瞬间替换为：亮度滑块（值为 100%）、对比度滑块（值为 50%）、伽马、黑电平等。
  - 焦点自动重置并高亮在最上方的“亮度”项上。
  - 按 Escape 键可完美回退到“图像”一级设置。
