# 产品规格说明书 (PRD) — 中国区灵悉 UI V5.2 菜单重构

## 1. 系统入口与页面架构
TCL OS 灵悉 UI V5.2 的设置系统分为两层入口：
1. **快捷控制中心 (Control Center)**：
   - 作为首层悬浮抽屉在屏幕左侧浮现。
   - 包含系统核心状态展示（时间、日期、天气）与高频调节项（亮度滑块、网络连接名、音频输出端口、蓝牙遥控器状态、环境感应 Toggle）。
   - 通过标签页与“全部设置”进行深层链接。
2. **系统设置面板 (Settings Activity)**：
   - 全局覆盖式级联菜单。
   - 包括：一栏左侧类别（Expanded/Collapsed 变宽交互）、二栏中间控制项列表、三栏右侧详情说明及视频层预览。

---

## 2. 核心还原组件设计规范

### 2.1 侧边栏收缩交互 (Sidebar Micro-Interaction)
- **展开态 (Expanded)**：当焦点位于第一列分类菜单上时，侧边栏宽度设为 `250px`，展现“图标 + 中文字符”。
- **折叠态 (Collapsed)**：当焦点向右移入中间选项列或右侧详情时，侧边栏自适应向左收缩至 `90px`，只留下图标居中显示，隐藏中文字符。这极大地拓展了主要设置项的水平展示范围。

### 2.2 信号源协议智能识别 (Dolby/SDR Tag)
- 图像菜单顶部包含一个动态识别当前影片信源编码的标识（如“SDR”或“Dolby Vision”）。
- 当图效模式切换为杜比（如“杜比视界IQ”）时，标识自动切换为“Dolby Vision IQ”；当图效为普通（如电影、标准、FILMMAKER）时，标识切换为“SDR”。

### 2.3 高级设置级联结构
中间栏支持点击后切换展示更深层的参数控制器组：
- **亮度控制组 (Brightness Group)**：
  - 亮度 (Slider): 0 - 100。
  - 对比度 (Slider): 0 - 100。
  - 伽马 (Picker): 1.8 / 2.2。
  - 自动 HDR 转换 (Toggle): 开/关。
  - 智能低蓝光护眼模式 (Toggle) 与 防蓝光强度滑块 (Slider)。
  - 黑电平 (Slider): 0 - 100。
  - 黑电平延伸 & 动态对比度 (Toggle)。
- **色彩控制组 (Color Group)**：
  - 饱和度 (Slider): 0 - 100。
  - 色调 (Slider): 0 - 100。
  - 色温 (Slider): 调节滑块，文字显示“冷、标准、暖”。
  - 色彩增强 (Toggle)。
  - 色彩空间 (Picker): 自动 / 原始。
