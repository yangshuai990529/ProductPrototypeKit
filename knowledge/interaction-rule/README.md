# 遥控器与交互规则 (Interaction Rules)

本目录规范了电视遥控器按键下的核心交互逻辑，确保 Demo 中的 JavaScript 焦点移动逻辑符合以下规则。

## 遥控器按键响应规则

1. **D-pad 导航**：
   - 移动：按 `ArrowUp`, `ArrowDown`, `ArrowLeft`, `ArrowRight` 后，系统焦点沿物理方向寻址，聚焦到最近的可聚焦元素。
   - 边界阻尼：当焦点移到列表或网格的最边缘时，再次按相同方向的按键，焦点应保持在当前位置，不能发生焦点丢失，且滚动区域应提供轻微的震动/回弹视觉反馈。

2. **确定键 (Enter/OK)**：
   - 触发激活：聚焦在按钮、开关或列表项上按 `Enter` 键时，触发相应的点击事件，执行功能激活、跳转或弹窗操作。

3. **返回键 (Back/Escape)**：
   - 逐步回退：按 `Escape` 必须关闭当前的最上层模态框（Dialog）、悬浮 Toast，或回退到上一级菜单。
   - 当没有可回退的层级时，应发出提示，告知用户是否退出应用。

4. **弹窗焦点锁定**：
   - 当展示确认/警告弹窗时，D-pad 焦点必须强制限制在弹窗内，按方向键仅在弹窗内的选项（如“确认”、“取消”）之间轮转。
